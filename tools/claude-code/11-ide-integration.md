## 11.2 VS Code

**安装:** 扩展市场搜索 "Claude Code"

**关键特性:**
- 内联 diff (接受/拒绝按钮)
- 多对话标签页管理
- `@-mentions` 引用文件和符号
- 自动共享编辑器上下文
- VS Code 1.109+ 原生兼容 CLAUDE.md、rules、agents、skills

**权限模式与 CLI 一致** — `Shift+Tab` 切换。

`--ide` 标志自动检测并连接 VS Code 或 JetBrains IDE。

## 11.3 JetBrains

- JetBrains 插件市场安装
- IDE 内代码补全与生成
- 项目上下文自动感知

## 11.4 桌面应用

- 独立窗口，无需终端或 IDE
- 支持计划模式、定时任务
- 订阅制
- 计划模式通过 `Ctrl+G` 触发

## 11.5 Web 模式

- 地址: [claude.ai/code](https://claude.ai/code)
- 远程仓库直接操作，不需克隆
- 内置多标签页
- 无 SSH 能力

## 11.6 平台选择建议

| 场景 | 推荐 |
|------|------|
| 日常开发 | VS Code 扩展 |
| CI/CD 流水线 | CLI `-p` 模式 |
| SSH 远程服务器 | CLI |
| 大型重构审查 | VS Code 或桌面 App (计划模式) |
| 快速查看远程仓库 | Web 模式 |
| JetBrains 用户 | JetBrains 插件 |
