# [最小交叉实验 — AI 拆解链上交易](https://aiweb3.school/zh/handbook/)

> Week 1 模块 C · 2026-05-20

**实验目的**：把模块 A（AI 基础）和模块 B（Web3 基础）连起来——用 AI 逐层拆解一笔真实的链上交易，验证每个链上字段对应哪个 Web3 概念。

## 实验过程

在 Sepolia 测试网发送 0.1 ETH 转账，获取交易哈希后交 AI Agent 解析 RPC 返回的原始数据。

**交易哈希**：`0xd7d486082ec118a9e9ec114e88a15abf4f00caea58b454ba43850c90d3b2f41f`

## 交易拆解

### 第一层：谁发给谁

```
FROM  0xb397d3d5d82f0128c76ffbf76cd50712c9ca0139
  │
  │  0.1 ETH (100,000,000,000,000,000 wei)
  ↓
TO    0xb9db1a922e97a06456d100d106320e006df68150
```

`input: 0x` → 没有调用任何合约函数，就是纯粹的 ETH 转账。

### 第二层：字段 → 概念映射

| 链上字段 | 数值 | 对应 Web3 概念 | 知识库 |
|----------|------|---------------|--------|
| Hash | `0xd7d4...` | 交易唯一指纹，改一个字符哈希完全不同 | 密码学 |
| r, s, v | 签名数据 | 私钥签名 + 公钥验证 | 密码学 |
| Nonce | 0 | 此地址第一笔交易，防重放 | 钱包 |
| Gas Limit | 31,500 | 愿意付的 gas 上限 | 钱包 |
| Gas Used | 21,000 | ETH 转账固定消耗 | 钱包 |
| Gas Price | ~2.9 Gwei | 每单位 gas 出价 | 钱包 |
| Chain ID | 11155111 | Sepolia 测试网 | 网络 |
| Type | 0x2 (EIP-1559) | 动态费用机制（非旧式固定 gas） | 网络 |
| Status | 0x1 | 交易执行成功 | 合约 |
| Block | #10886427 | 被打包进这个区块 | 网络 |

### 第三层：交易生命周期

```
① MetaMask 点确认
      │
      ▼
② 本地签名（私钥不出设备，只生成 r,s,v）
      │
      ▼
③ 广播到 Sepolia mempool（待打包区）
      │
      ▼
④ 验证节点挑中 → 打包进区块 #10886427
      │
      ▼
⑤ EVM 执行：from 扣 0.1 ETH + gas，to 加 0.1 ETH
      │
      ▼
⑥ 状态写入链上，永久可查
```

### 第四层：如果 AI Agent 要替代人

如果让 AI Agent 自动执行类似转账，至少需要：

- **额度限制**：不能无限转账
- **地址白名单**：只能转到可信地址
- **交易模拟**：签名前预演结果
- **人工确认**：高风险动作保留 human-in-the-loop
- **审计记录**：每次执行留日志
